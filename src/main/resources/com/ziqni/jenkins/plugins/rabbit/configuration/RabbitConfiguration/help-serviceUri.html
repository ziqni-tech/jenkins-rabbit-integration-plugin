You can set the URI for the RabbitMQ service you want to connect to. The URI should follow the AMQP URI specification. (Note: you cannot specify the `amqp_userinfo` part.)

If you do not provide a username/password, the following default values will be used:

  • Username: guest
  • Password: guest

Note that any existing connection will be disconnected if the URI, username, or password is changed while connecting, or if left empty.

You can test the connection using the test button below. This will not affect any existing connection.
