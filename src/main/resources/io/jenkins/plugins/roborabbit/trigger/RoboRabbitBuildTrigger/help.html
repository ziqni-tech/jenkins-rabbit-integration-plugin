Robo Rabbit AMQP Build Trigger

This trigger listens to messages from RabbitMQ queues and starts a build when a relevant message is received.

Configuration:

  • Connection URI: The URI to connect to the RabbitMQ instance. This should point to a RabbitMQ server.
  • Queue Name: The name of the queue to listen for messages that will trigger the build.
  • Routing Key: (Optional) Specify the routing key for finer control over message filtering.

The trigger can be used to automate builds based on messages published to a specific queue.
